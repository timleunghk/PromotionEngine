# PromotionEngine

This Project Contains Code and Test of Promotion Engine using Visual Studio 2019 by .NET 5.0 

## Folder Description

ClassLibraryPromotionEngine -- Product Data, Promotion Plan, Calculator
ConsoleTestPromotionEngine -- Console Application, for user to entry quantity of items to buy
UnitTestForPromotionEngine -- Unit Test Scripts to check methods in ClassLibraryPromotionEngine is passed or failed
